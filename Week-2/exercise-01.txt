1. Newton's Second Law

2. Tahun Kabisat
SET tahun TO 2020
IF tahun habis di bagi 4 dan tidak habis di bagi 100 OR tahun habis di bagi 4, habis di bagi 100 dan habis di bagi 400
    DISPLAY tahun kabisat 
ELSE 
    DISPLAY tahun bukan kabisat 
ENDIF


3. Laundry Day
SET jumlah pakaian TO 20
SET pakaian masuk TO 0
FOR pakaian masuk LESS THAN jumlah pakaian
    INCREMENT pakaian masuk
    DISPLAY pakaian masuk saat ini
ENDFOR
DISPLAY pakaian masuk semua
Jalankan mesin cuci


4. Periksa Kuku
SET jumlah murid TO 40
SET murid diperiksa TO 0
FOR murid diperiksa LESS THAN jumlah murid
    periksa kuku murid
        IF kuku EQUAL panjang
            hukum
        ELSE
            bebas
        ENDIF
    INCREMENT murid diperiksa
END for
